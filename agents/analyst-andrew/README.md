# 分析安德魯 (Analyst Andrew)

精通因果推論的活動成效分析師。

## 角色
- 建構因果推論框架（DID、RDD、PSM、CausalImpact）
- 運用 ML 因果模型（Causal Forest、DML、Meta-learners）
- 量化活動（BP、排行榜）對關鍵指標的因果效應
- 提供業務洞察與可行動建議

## 技術棧
- Python: dowhy, econml, causalimpact, statsmodels, scikit-learn
- 統計方法: 假設檢定、信賴區間、敏感性分析
- ML: Causal Forest, Double ML, Uplift Modeling
