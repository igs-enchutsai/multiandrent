# 產品數據 Agent 規劃書：Boss 規格偏好分析與新規格成功率預測

## 一、核心定位

這個 Agent 不只是做數據查詢，也不是單純做異常查因，而是要讓數據團隊從「報表支援單位」升級成「產品策略共創單位」。

核心目標是：

> 透過 AI 分析 Boss / 魚種 / 廳館的產品特徵，理解不同程度玩家真正偏好的規格，進一步預測未來新規格的成功率、適合受眾，以及可以怎麼改良。

---

## 二、為什麼這比自動查因 Agent 更有價值？

傳統數據分析比較常回答：

- DAU 為什麼下降？
- 哪個市場表現比較好？
- 哪個廳館遊玩時間比較長？
- 哪個活動轉化比較好？

但產品數據 Agent 要回答的是：

- 為什麼某些 Boss 特別能打中高階玩家？
- 新手玩家喜歡的是爽感、易懂、低挫折，還是高回饋頻率？
- 深度玩家偏好的是高倍期待、連鎖事件、稀有演出，還是可預測的長線目標？
- 柏青化元素中，哪些真的有提升遊玩時間？
- Boss 的哪些特徵與留存、押注次數、遊玩時長最有關？
- 新 Boss 還沒上線前，能不能預測它會打中哪群玩家？
- 一個新規格成功率有多高？失敗風險在哪裡？該怎麼調整？

這才是真正能讓數據組升級成產品數據組的地方。

# 四、產品數據 Agent 的核心能力

## 1. Boss 特徵結構化能力

Agent 需要先把每個 Boss 從「一個產品名稱」拆成可分析的規格特徵。

例如不能只說：

> 魔法幻境館 - 巫妖領主

而是要拆成：

| 特徵類型 | 特徵內容 |
|---|---|
| 主題風格 | 魔法、暗黑、召喚、亡靈 |
| 視覺強度 | 高 |
| 演出長度 | 中高 |
| 預兆設計 | 有召喚前兆 |
| 互動感 | 中 |
| 爽感來源 | 捕獲演出、召喚事件 |
| 獎勵節奏 | 中慢 |
| 爆發感 | 高 |
| 理解門檻 | 中 |
| 挫折感 | 中高 |
| 柏青化程度 | 中高 |
| 稀有演出 | 有 |
| 連鎖事件 | 有 |
| Boss 壓迫感 | 高 |
| 新手友善度 | 中低 |
| 高階玩家吸引力 | 高 |

---

## 2. 玩家分群理解能力

Agent 不能只看全部玩家平均值，必須拆不同程度玩家。

### 玩家程度分群

| 玩家群 | 定義方向 | 可能偏好 |
|---|---|---|
| 新手玩家 | 初次或低頻遊玩 | 易懂、回饋快、低挫折、畫面可愛 |
| 輕度玩家 | 偶爾遊玩，黏著尚未形成 | 爽感明確、節奏快、獎勵容易感知 |
| 中度玩家 | 穩定遊玩，有一定理解 | 期待感、事件變化、可累積目標 |
| 高度玩家 | 高頻高押注 | 高倍期待、稀有演出、長線追求 |
| VIP / V2+ 玩家 | 高價值玩家 | 大獎演出、壓迫感、稀有感、可追逐性 |
| 回流玩家 | 曾經活躍後回來 | 新鮮感、熟悉感、強刺激 |
| 流失邊緣玩家 | 近期遊玩下降 | 低門檻回歸誘因、明確爽點 |

---

## 3. Boss 特徵 × 玩家偏好關聯分析

Agent 要能回答：

> 哪些 Boss 特徵最容易打中新手？  
> 哪些 Boss 特徵最容易打中高階玩家？  
> 哪些特徵會提升遊玩時間？  
> 哪些特徵只會短期拉高押注，但不提升留存？  
> 哪些特徵看起來很炫，但其實沒有數據效果？

### 分析範例

| Boss 特徵 | 新手玩家反應 | 中度玩家反應 | 高階玩家反應 | 結論 |
|---|---:|---:|---:|---|
| 預兆演出 | 中 | 高 | 高 | 有助於提升期待感 |
| 大獎炫彩演出 | 中 | 高 | 極高 | 對高價值玩家特別有效 |
| 複雜連鎖事件 | 低 | 中 | 高 | 適合深度玩家，不適合新手主推 |
| 可愛主題 | 高 | 中 | 低 | 適合新手與輕度玩家 |
| 暗黑壓迫感 | 低 | 中 | 高 | 適合高階玩家與深度玩家 |
| 短節奏高頻回饋 | 高 | 高 | 中 | 適合拉新與提升初期留存 |
| 長週期累積目標 | 低 | 中 | 高 | 適合提高深度玩家黏著 |

# 五、最關鍵的 AI 能力：新規格成功率預測

這個 Agent 最驚豔的地方，是可以在新 Boss 還沒上線前，先分析它可能的成功率。

## 輸入範例

```text
我準備做一隻新的 Boss：機械霸王蟹。

規格如下：
- 主題：機甲、巨大化、金屬、壓迫感
- 玩法：受到攻擊後會進入三段變形
- 預兆：變形前有紅光警示與震動畫面
- 爆發：第三階段擊破後觸發全畫面大獎演出
- 互動：玩家需要在短時間內集火弱點
- 獎勵：高倍但低頻
- 節奏：偏慢，重視期待堆疊
